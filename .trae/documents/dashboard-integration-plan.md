# 仪表盘前端和后台集成计划

## 1. 前端API模块创建

### 1.1 创建仪表盘API模块
- **文件**：`src/api/dashboard.ts`
- **功能**：封装与后端仪表盘相关的API调用
- **接口**：
  - `getDashboardStats()`：获取仪表盘统计数据
  - `getTodoList()`：获取待办事项列表
  - `processTodo(id)`：处理待办事项
  - `getTodoDetail(id)`：获取待办事项详情
  - `getNoticeList()`：获取系统公告列表
  - `getNoticeDetail(id)`：获取公告详情
  - `getIncomeExpenseData()`：获取收支趋势数据
  - `getBusinessTypeData()`：获取业务类型分布数据
