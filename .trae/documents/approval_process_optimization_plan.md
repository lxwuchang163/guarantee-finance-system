# 审批流程优化实施计划

## 1. 需求分析

根据用户需求，需要完成以下任务：
1. 审批流程的流程模板放到基础管理作为二级菜单
2. 菜单权限的菜单管理和角色权限配置管理按照目前菜单结构配置
3. 角色管理进行优化，根据新的菜单情况
4. 调整我的审批和我发起审批的功能

## 2. 现状分析

### 当前文件结构
- **前端**：
  - `src/views/system/menu/index.vue` - 菜单管理（包含菜单管理和角色权限配置）
  - `src/views/system/role/index.vue` - 角色管理
  - `src/views/system/process/index.vue` - 审批流程管理（包含流程模板、我的审批、我发起的）

- **后端**：
  - `SysMenuController.java` - 菜单管理控制器
  - `SysRoleController.java` - 角色管理控制器
  - `ProcessServiceImpl.java` - 审批流程服务实现

### 当前功能状态
- 审批流程模板当前位于系统管理下
- 菜单管理和角色权限配置已在当前菜单结构中实现
- 角色管理功能完整，但需要根据新的菜单结构进行优化
- 审批流程包含流程模板、我的审批、我发起的三个部分

## 3. 实施计划

### 3.1 步骤一：创建基础管理菜单结构

#### 前端修改
1. **修改菜单配置**：
   - 在菜单管理中创建基础管理目录
   - 将审批流程模板移动到基础管理下作为二级菜单

2. **创建基础管理相关文件**：
   - 创建 `src/views/basic/process-template/index.vue` - 流程模板管理页面
   - 创建 `src/views/basic/index.vue` - 基础管理主页面

#### 后端修改
1. **更新菜单数据**：
   - 在数据库中添加基础管理菜单
   - 将审批流程模板菜单移动到基础管理下

### 3.2 步骤二：调整审批流程功能

#### 前端修改
1. **拆分审批流程页面**：
   - 将 `src/views/system/process/index.vue` 中的流程模板部分移除
   - 保留并优化 "我的审批" 和 "我发起的" 功能
   - 创建 `src/views/system/approval/index.vue` - 审批管理页面（包含我的审批和我发起的）

2. **更新路由配置**：
   - 更新 `src/router/index.ts`，添加基础管理路由
   - 调整审批相关路由结构

#### 后端修改
1. **调整API接口**：
   - 确保审批流程相关接口支持新的菜单结构
   - 优化审批查询和操作接口

### 3.3 步骤三：优化角色管理

#### 前端修改
1. **更新角色管理页面**：
   - 修改 `src/views/system/role/index.vue`，确保与新的菜单结构适配
   - 优化角色权限配置界面，支持新的菜单层级

2. **更新菜单管理页面**：
   - 确保 `src/views/system/menu/index.vue` 中的角色权限配置支持新的菜单结构

#### 后端修改
1. **更新角色权限逻辑**：
   - 确保角色权限分配逻辑与新的菜单结构匹配
   - 优化角色权限查询接口

### 3.4 步骤四：测试和验证

1. **功能测试**：
   - 测试流程模板管理功能
   - 测试我的审批和我发起的功能
   - 测试角色管理和权限配置功能

2. **集成测试**：
   - 验证所有功能模块的集成
   - 确保系统正常运行

## 4. 技术实现要点

### 4.1 前端实现
- 使用 Vue 3 Composition API
- 保持 Element Plus 组件风格一致
- 确保路由配置正确
- 实现响应式布局

### 4.2 后端实现
- 确保菜单数据结构正确
- 优化API接口性能
- 保持权限控制逻辑一致

### 4.3 数据库实现
- 更新菜单表结构（如果需要）
- 确保菜单数据迁移正确

## 5. 风险评估

### 5.1 潜在风险
- 菜单结构变更可能影响现有权限配置
- 流程模板移动可能影响现有流程实例
- 路由变更可能导致前端页面访问异常

### 5.2 风险缓解措施
- 提前备份数据库
- 测试所有功能模块
- 确保权限配置正确迁移
- 提供回滚方案

## 6. 预期成果

- 审批流程模板成功移动到基础管理下
- 菜单权限管理和角色权限配置正常运行
- 角色管理功能与新的菜单结构适配
- 我的审批和我发起的功能优化完成
- 系统整体运行稳定
