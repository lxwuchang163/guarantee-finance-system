# 银企直连和银行对账表格样式统一计划

## 目标
将银企直连和银行对账的表格样式修改为与付款单管理保持一致，确保整个系统的UI风格统一。

## 现状分析

### 付款单管理（参考样式）
- **表格属性**：`stripe border v-loading="loading"`
- **表格宽度**：自动适应
- **固定列**：左侧和右侧固定列
- **操作列宽度**：240px
- **分页**：完整的分页控件，位于表格下方右侧
- **搜索区域**：内联表单，包含关键词、业务类型、状态、日期范围等筛选条件
- **卡片头部**：包含标题和操作按钮
- **样式**：标准的Element Plus样式，无size属性

### 银企直连（需要修改）
- **表格属性**：`stripe border v-loading="tableLoading" size="small"`
- **表格宽度**：固定样式 `style="margin-top: 12px;"`
- **固定列**：只有右侧固定列
- **操作列宽度**：200px
- **分页**：无分页
- **搜索区域**：无标准搜索区域，只有操作按钮栏
- **卡片头部**：只包含标题
- **样式**：使用了size="small"，与参考样式不一致

### 银行对账（需要修改）
- **表格属性**：`stripe border v-loading="tableLoading" size="small"`
- **表格宽度**：固定样式 `style="margin-top: 12px;"`
- **固定列**：只有右侧固定列
- **操作列宽度**：100px
- **分页**：有分页，但样式可能与参考样式不一致
- **搜索区域**：有搜索区域，但样式与参考样式不一致
- **卡片头部**：只包含标题
- **样式**：使用了size="small"，与参考样式不一致

2. **表格宽度**：移除 `style="margin-top: 12px;"`，使用默认布局

- **缓解措施**：根据实际数据情况，适当调整列宽，确保数据显示完整
